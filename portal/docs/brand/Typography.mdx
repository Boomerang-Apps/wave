# Typography

WAVE uses two typefaces: **Inter** for all UI text and **JetBrains Mono** for code, terminal output, and metrics. Both are loaded via Google Fonts and defined in the Tailwind config.

---

## Sans Serif -- Inter

The primary typeface for headings, body text, labels, and navigation. Inter is optimized for screen readability at small sizes, making it ideal for data-dense dashboards.

**Tailwind class:** `font-sans`
**Stack:** `'Inter', 'system-ui', 'sans-serif'`

## Monospace -- JetBrains Mono

Used for code snippets, terminal output, file paths, commit hashes, metrics, and any machine-readable content. JetBrains Mono features programming ligatures and clear character distinction.

**Tailwind class:** `font-mono`
**Stack:** `'JetBrains Mono', 'monospace'`
