# Color System

WAVE uses HSL-based CSS custom properties with a class-based dark mode toggle (`.dark`). All Tailwind color utilities reference these tokens via `hsl(var(--name))`, ensuring automatic theme switching.

The palette is inspired by **Linear's dark mode** -- cool-tinted near-black backgrounds with subtle blue undertones, an **indigo brand accent** (`#5e6ad2`), and semantic status colors (green, yellow, red, blue).
