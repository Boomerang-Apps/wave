# Spacing and Layout

WAVE's layout system uses Tailwind's default spacing scale, a `--radius` CSS custom property for consistent border rounding, and the `tailwindcss-animate` plugin for motion.

---

## Border Radius Scale

The `--radius` custom property is set to `0.5rem` (8px). All radius tokens derive from this base value.

## Spacing Quick Reference

<div style={{ padding: '1.5rem', borderRadius: '0.5rem', border: '1px solid #e5e5e5' }}>

```
Tailwind Scale (commonly used values):

0.125rem =  2px  (0.5)   -- item gap in tight lists
0.25rem  =  4px  (1)     -- inline spacing, icon margins
0.5rem   =  8px  (2)     -- gap between inline elements, button icon gap
0.75rem  = 12px  (3)     -- stack gap, vertical rhythm
1rem     = 16px  (4)     -- card grid gap, form field spacing
1.5rem   = 24px  (6)     -- card padding, section gap
2rem     = 32px  (8)     -- page padding, large section gap
2.5rem   = 40px  (10)    -- major section dividers
3rem     = 48px  (12)    -- page-level vertical spacing
```
