# Component Inventory

WAVE's component library is organized into seven categories. **Primitives** are built on Radix UI and live in `src/components/ui/`. **Feature components** live in `src/components/` and compose primitives with domain-specific logic.

All components consume the HSL-based theme tokens, respond to `.dark` class toggling, and use `tailwindcss-animate` for motion.

---
